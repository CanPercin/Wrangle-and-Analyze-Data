Insights and Visualizations¶

WeRateDogs is a twitter account that posts the pictures of dogs with humorous comment and ratings. These posts include images that are shared from dog owners, ratings related with the humorous content, types, names and feature of their stages like puppo / doggo etc.

In this project gathering, assessing, storing and demonstrating skills are presented. After creating the data set, some visualization and insights are presented.

1 - Highest retweeted post is a video of a labrador retriever (doggo) standing in the pool¶

import requests

class Tweet(object):
    def __init__(self, s, embed_str=False):
        if not embed_str:
            api = 'https://publish.twitter.com/oembed?url={}'.format(s)
            response = requests.get(api)
            self.text = response.json()["html"]
        else:
            self.text = s

    def _repr_html_(self):
        return self.text

Tweet("https://twitter.com/dog_rates/status/744234799360020481")
